Physical Prototype

Hardware Components

Raspberry Pi 3B+: Central processing unit running the AI algorithms and coordinating all components.
Camera Module: Captures real-time video feed of traffic for vehicle detection and classification using YOLO.
LED Arrays: Represent traffic signals at the intersection, controlled via GPIO pins on the Raspberry Pi.
Wireless Communication Module: Enables coordination between multiple intersections using MQTT protocol.

Software Components

YOLOv5 TensorFlow Lite: Optimized for Raspberry Pi to perform real-time vehicle detection with 5 vehicle classes.
