Objectives & Scope

Main Objective

Design and implement an AI-driven real-time traffic control system that improves traffic flow in a network of roads through real-time smart data-driven decisions.

Specific Objectives

Develop a SUMO-based simulation with 20+ routes and interconnected intersections.
Implement AI algorithms for traffic signal control based on real-time data.
Integrate communication protocols for coordination between intersections.

Project Scope

Simulation of a section of Nairobi city with 100+ intersections where vehicles pass through an average of five intersections per route.
Physical prototype using Raspberry Pi 3B+ and YOLO-based image detection for vehicle classification.
Real-time traffic density data collection and analysis using AI tools for smarter decisions.
Exclusions: No integration of complex sensor systems or real-time data collection from actual vehicles. Physical model limited to small-scale demonstration.
