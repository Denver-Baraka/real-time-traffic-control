System Architecture

5-Layer Architecture

1
Sensor Layer: Traffic cameras with YOLO object detection, loop detectors, and environmental sensors collect raw traffic data.
2
Data Layer: Handles data collection, storage (locally on Raspberry Pi's SD card), and preprocessing for analysis.
3
AI Layer: Core intelligence with traffic prediction models, signal timing optimization, dynamic routing, and incident detection.
4
Control Layer: Implements decisions through adaptive traffic signals via Raspberry Pi and variable message signs.
5
Management Layer: Provides centralized monitoring dashboard and integration with emergency services.
